\doxysubsection{Detailed Description}
This file defines the \mbox{\hyperlink{classNaluPacketParser}{Nalu\+Packet\+Parser}} class responsible for parsing Nalu packets from a byte stream. 

The \mbox{\hyperlink{classNaluPacketParser}{Nalu\+Packet\+Parser}} class is used to process raw byte streams, identify valid Nalu packets, and store them in a vector of \mbox{\hyperlink{classNaluPacket}{Nalu\+Packet}} structures. The class includes functionality to validate packet integrity with start and stop markers, extract relevant data from the byte stream, and handle any leftover or incomplete packets.

The class is configurable with several parameters such as packet size, marker values, and timing masks.

\begin{DoxyNote}{Note}
The parsing process can be customized by setting flags for checking packet integrity. 
\end{DoxyNote}
