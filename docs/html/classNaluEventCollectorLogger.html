Detailed Description

A logger class for the NaluEventCollector.

This class provides logging functionality with different log levels (DEBUG, INFO, WARNING, ERROR), allowing logs to be output to both the console (with optional color formatting) and a log file. The logging behavior can be customized by adjusting the log level, enabling/disabling file logging, and changing the color scheme for each log level.

enum NaluEventCollectorLogger::LogLevel

Enumeration for log levels.

The log levels determine the verbosity of the logs:

  • DEBUG: Detailed information, usually for debugging.
  • INFO: General information messages.
  • WARNING: Warnings about potential issues.
  • ERROR: Errors that indicate failures or critical issues.
