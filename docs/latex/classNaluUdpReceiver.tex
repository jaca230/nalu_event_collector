\doxysubsection{Detailed Description}
A class for receiving UDP packets, parsing them, and storing the data in a buffer. 

This class provides functionality to initialize a UDP socket, receive packets, and store them in a buffer for further processing. It runs in a separate thread to continuously receive data from the network.
