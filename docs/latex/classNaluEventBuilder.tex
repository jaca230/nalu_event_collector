\doxysubsection{Detailed Description}
Class for building events from Nalu packets. 

The {\ttfamily \mbox{\hyperlink{classNaluEventBuilder}{Nalu\+Event\+Builder}}} class is responsible for collecting events from Nalu packets. It manages the event buffer, handles the safety buffer zone, and provides methods for collecting events and interacting with the internal event buffer.
