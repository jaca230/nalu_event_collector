Detailed Description

Represents a single NALU event consisting of multiple NALU packets.

The NaluEvent class stores the header, information, reference time, and a collection of NaluPackets associated with a single event. It supports adding packets, serializing the event data, and checking if the event is complete based on predefined windows and channels.
