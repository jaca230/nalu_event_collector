Detailed Description

A thread-safe buffer for storing raw UDP packet data.

This class provides methods to append data to the buffer, pop individual bytes, retrieve all bytes, and wait for the buffer to contain a minimum number of bytes. It also supports overflow notifications and checks if the buffer is empty or full.
