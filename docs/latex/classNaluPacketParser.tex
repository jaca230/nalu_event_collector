\doxysubsection{Detailed Description}
A class for parsing Nalu packets from a byte stream. 

The \mbox{\hyperlink{classNaluPacketParser}{Nalu\+Packet\+Parser}} processes byte streams, identifies Nalu packets based on specified start and stop markers, and extracts relevant data into \mbox{\hyperlink{classNaluPacket}{Nalu\+Packet}} structures. It supports both marker-\/based integrity checks and raw parsing for custom use cases.
