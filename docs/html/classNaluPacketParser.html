Detailed Description

A class for parsing Nalu packets from a byte stream.

The NaluPacketParser processes byte streams, identifies Nalu packets based on specified start and stop markers, and extracts relevant data into NaluPacket structures. It supports both marker-based integrity checks and raw parsing for custom use cases.
